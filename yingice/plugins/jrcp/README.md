# jrcp

## 插件描述

群聊今日 CP 插件。插件会查询当前群成员列表，并以当前日期和群号作为随机种子，为群成员生成当天稳定的 CP 配对。

## 指令说明

仅在群聊中触发：

```text
。jrcp / .jrcp / !jrcp
```

回复格式：

```text
张三（123456）的今日cp为：李四（654321）【李四的头像】
```

## 其他内容

- 群成员数为偶数时，bot 自己也参与配对；群成员数为单数时，bot 自己不计入配对池。
- 每次触发都会重新获取群成员列表。
- 群名片变化会实时反映在输出中。
- 群成员进出会影响当天后续查询结果，因为当前版本没有保存每日配对缓存。
- 输出中的头像来自 QQ 头像地址。
