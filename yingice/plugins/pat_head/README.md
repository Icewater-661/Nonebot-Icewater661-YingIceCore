# pat_head

## 插件描述

摸头互动插件。用户 @bot 并发送摸头指令后，bot 会回复互动文案，并调用好感度系统尝试增加好感。

## 指令说明

```text
@bot 摸头
```



## 其他内容

回复示例：

```test
唔...被摸头了，感觉有点开心。 
好感上升了1，当前好感：101
```

当天已经增加过正向好感时：

```test
唔...被摸头了，感觉有点开心。 
当前好感：101
```



- 每次摸头尝试增加 `1` 点好感。
- 正向好感每天最多生效一次。
- 好感变更逻辑由 `ying_point_system` 插件提供。
